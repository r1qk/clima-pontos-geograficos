### Objetivo da Sprint
Desenvolver uma solução para monitoramento remoto de áreas de vegetação sujeitas a riscos de eventos extremos.

### Descrição da Atividade
Desenvolver um dashboard utilizando dados fictícios de sensoriamento aplicados a uma região específica.

A solução deverá contemplar:

Utilização de uma API pública (ex.: OpenStreetMap) para obter pelo menos 10 pontos geográficos, contendo latitude e longitude;
Exibição desses pontos no dashboard;
Integração com uma API meteorológica (ex.: Open-Meteo) para obtenção das condições climáticas de cada ponto;
Apresentação das informações meteorológicas no dashboard de forma clara e organizada.
