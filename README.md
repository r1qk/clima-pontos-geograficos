# Grovis - Obtendo informações climáticas do Pantanal
## Objetivo
Uma empresa fictícia busca formas de monitorar áreas sujeitas a risco. Esse projeto
busca informar ao usuário as informações climáticas sobre lugares do Pantanal.

### Vídeo
Link do vídeo: https://www.youtube.com/watch?v=opWUsmIZW2s

### APIs utilizadas
- OpenStreetMap;
- OpenMeteo.

### Exemplo de dado utilizado
Link da API (OpenStreetMap) para buscar latitude e longitude em Corumbá: https://nominatim.openstreetmap.org/search?q=Corumb%C3%A1%2C%20Brasil&format=json&limit=1  

Link da API (OpenMeteo) para buscar informações climáticas de Corumbá: https://api.open-meteo.com/v1/forecast?latitude=-19.0006027&longitude=-57.6507535&current=temperature_2m,relative_humidity_2m,wind_speed_10m
